#### Servir localmente

## **Clone** o repositório

```bash
git clone https://github.com/WesleyHernandes/star-wars.git
```

## Setup
Acesse o diretório clonado e realize a instalação das dependências:

```bash
# npm
npm install

# pnpm
pnpm install

# yarn
yarn install

# bun
bun install
```

## Inicie o servidor

```bash
# npm
npm run dev

# pnpm
pnpm run dev

# yarn
yarn dev

# bun
bun run dev
```

Servidor de desenvolvimentos estará iniciado em: `http://localhost:3000`:
